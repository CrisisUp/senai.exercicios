# 📘 Atividade 19: O Guardião Universal (Introdução a Templates)
STATUS: Nível 19 (Guardião) - Trilha POO Mastery

## 🎯 Estudo de Caso
A Central de Comando Nexus agora gerencia não apenas Drones, mas também pacotes de carga (`Cargo`). Você precisa de uma classe de "Logística" que possa transportar qualquer coisa. Em vez de criar uma classe `LogisticaDrone` e outra `LogisticaCargo`, você aprenderá a usar **Templates** para criar uma classe universal que funciona com qualquer tipo de dado, mantendo a segurança de tipos (Type Safety).

## 🛠️ Necessidade Técnica
- **`template <typename T>`**: Define um parâmetro de tipo genérico.
- **`T`**: O placeholder que será substituído pelo tipo real no momento do uso.

### Exemplo (C++):
```cpp
template <typename T>
class Hangar {
private:
    T* item;
public:
    void guardar(T* novo) { item = novo; }
    T* obter() { return item; }
};

// Uso de Elite:
Hangar<Drone> hangarDrones;
Hangar<Sensor> hangarSensores;
```

## 🚀 Missão
1.  Crie a classe template `CaixaDeTransporte<T>`.
2.  Implemente métodos para guardar e retirar um item do tipo `T`.
3.  No `main()`, use a mesma classe para transportar um `int` (peso) e depois um objeto `Drone`.
4.  Compreenda como o compilador gera versões específicas da sua classe para cada tipo.

## 📋 Requisitos de Elite
- Reuso máximo de código através da generalização.
- Garantir a integridade de tipos mesmo em classes genéricas.
- Entender que templates são processados em tempo de compilação (zero overhead de runtime).
