# 📘 Atividade 05: A Inteligência Coletiva (Membros Estáticos)
STATUS: Nível 05 (Escultor) - Trilha POO Mastery

## 🎯 Estudo de Caso
A Central de Comando da "SkyCargo Nexus" precisa saber, em tempo real, quantos drones estão ativos no mundo todo. Atualmente, cada drone sabe apenas da sua própria bateria e modelo. Você aprenderá a usar **Membros Estáticos** (`static`): atributos e métodos que pertencem à **Classe** inteira, e não a um objeto específico. É como um "contador global" compartilhado por toda a frota.

## 🛠️ Necessidade Técnica
- **`static` Atributo:** Uma variável que tem apenas uma cópia na memória, compartilhada por todas as instâncias da classe.
- **`static` Método:** Uma função que pode ser chamada sem precisar instanciar um objeto (ex: `Drone::getTotalDrones()`).

### Exemplo (C++):
```cpp
class Drone {
private:
    static int totalDrones; // Declaração
public:
    Drone() { totalDrones++; } // Incrementa no nascimento
    static int getTotal() { return totalDrones; }
};

int Drone::totalDrones = 0; // Inicialização obrigatória fora da classe
```

## 🚀 Missão
1.  Na classe `Drone`, adicione um atributo privado estático `totalDrones`.
2.  No construtor, incremente esse contador.
3.  No destrutor, decremente o contador (simulando a saída de serviço).
4.  Crie um método estático público `exibirFrota()` que imprima a contagem atual.
5.  No `main()`, crie 3 drones, exiba a frota, destrua um deles (usando escopo `{ }`) e exiba a frota novamente.

## 📋 Requisitos de Elite
- Inicializar o membro estático corretamente no escopo global (fora da classe).
- Refletir o nascimento e a morte de objetos no contador global.
- Chamar o método estático usando a sintaxe de classe (`Classe::metodo`).
