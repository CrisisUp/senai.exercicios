# 📘 Protocolo SQL Mastery (PEC-SQL)

Este documento guia o aprendizado de Banco de Dados.

## 🎯 Objetivo

Transformar o aluno em um Arquiteto de Dados capaz de modelar e extrair inteligência de grandes volumes de informação.

## 📋 Padrão de Entrega OBRIGATÓRIO

Toda atividade deve conter:

1. **README.md:** Estudo de caso e requisitos.
2. **queries.sql:** Scripts de criação e população.
3. **INTERACAO_SQLITE.md:** Guia de "Copiar e Colar" para o terminal interativo (sem números de linha).
