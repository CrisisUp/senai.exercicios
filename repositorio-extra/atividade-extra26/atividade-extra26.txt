DOMÍNIO: Finanças Internacionais / Investimentos
ATIVIDADE: Extra 26
CONCEITO NOVO: Function Templates (Programação Genérica)

1. O CENÁRIO
Uma corretora de valores opera com diferentes tipos de precisão financeira. 
Alguns cálculos de conversão de moeda usam 'int' (para moedas sem decimais 
ou centavos), outros usam 'float' para rapidez, e os mais críticos usam 
'double' ou 'long double' para precisão bancária total.

2. A NECESSIDADE TÉCNICA
Até agora, se quiséssemos criar uma função 'multiplicarPorTaxa', teríamos que 
criar três versões: uma para int, uma para float e uma para double. Isso gera 
duplicação de código.

Os TEMPLATES permitem criar um "molde" de função. Você escreve o código uma 
única vez usando um tipo genérico 'T', e o compilador C++ gera automaticamente 
a versão correta para cada tipo de dado que você usar.
- Sintaxe: template <typename T>
- Vantagem: Código 100% reutilizável e tipagem segura.

3. REQUISITOS DO SISTEMA
- Criar um Template de Função 'aplicarTaxa(valor, taxa)'.
- O template deve funcionar para qualquer tipo numérico.
- Demonstrar o uso da mesma função template para:
    * Converter Iene (int) para Real.
    * Converter Dólar (double) para Real.
    * Somar saldos de diferentes tipos usando um template de soma.
- Interface colorida para distinguir os tipos de dados.
