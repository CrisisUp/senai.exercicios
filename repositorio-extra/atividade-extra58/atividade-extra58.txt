# 📘 Atividade Extra 58: Templates de Funções e Especialização (Análise de Precisão)

## 🎯 Estudo de Caso
A fábrica "SmartMatrix" precisa comparar leituras de diversos tipos de sensores: Pressão (`double`), Temperatura (`int`) e Identificadores de Lote (`string`). Embora a comparação padrão (`a < b`) funcione para números, comparar strings de IDs requer uma lógica diferente (ignorar maiúsculas/minúsculas) e comparar objetos complexos (como `LeituraSensor`) exige verificar múltiplos campos.

## 🛠️ Necessidade Técnica
No **Nível 31 (Engenheiro de Elite)**, aprendemos que nem todo template deve ser genérico para sempre.
*   **Function Templates:** Permitem criar uma única função que aceita qualquer tipo.
*   **Template Specialization (Especialização):** Permite criar uma versão específica de uma função genérica para um tipo de dado que exige um tratamento especial.
*   **Sobrecarga de Templates:** O compilador escolhe a versão mais específica disponível, garantindo a maior eficiência e correção possível.

## 📋 Requisitos (Arquitetura Modular)
1. Criar um template de função `comparar(T a, T b)` que retorna o maior valor.
2. Criar uma **Especialização de Template** para o tipo `std::string` que compara o comprimento das strings em vez da ordem alfabética.
3. Criar uma struct `LeituraSensor` e demonstrar como o template genérico falha se não houver especialização ou sobrecarga de operadores.
4. Criar `Comparador.h` e `atividade-extra58-templates.cpp`.
