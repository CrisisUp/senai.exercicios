ESPECIFICAÇÃO DE ENGENHARIA (WEB WORKERS)

Diferente do async/await (que ainda roda no thread principal), o Web Worker aloca uma fatia de processamento real do SO (Thread). É a única forma de garantir que algoritmos pesados não causem Jank na interface do operador.
