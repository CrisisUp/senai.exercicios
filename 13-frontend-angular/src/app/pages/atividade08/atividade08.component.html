Nota Técnica: Services e Injeção de Dependência

Nesta atividade, elevamos a arquitetura para o nível Arquiteto. Em vez de manter o intervalo e os dados no componente, usamos um Service Singleton (FleetService). O componente utiliza a função inject() para obter os dados. O Angular garante que se você navegar para outra página, o componente será destruído, mas o serviço continuará mantendo o estado global se desejado, ou pode ser limpo via OnDestroy.
